# Continuous Integration

Help us improve this article. Write how did you set up CodeceptJS for CI and see your post listed here!

Continuous Integration services allows you to delegate the control of running tests to external system. CodeceptJS plays well with all types of CI even when there is no documentation on this topic, it is still easy to set up with any kind of hosted or cloud CI. Our community prepared some valuable recipes for setting up CI systems with CodeceptJS.
